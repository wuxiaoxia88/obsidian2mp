# MP Publisher - Obsidian 微信公众号发布插件

一款 Obsidian 插件，支持为文章生成封面图、排版转换和一键发布到微信公众号草稿箱。

## 功能

### 1. 封面图生成
- 基于文章标题自动生成 **2.35:1** 比例封面图
- 6 种视觉风格：渐变流光、几何图形、极简风格、流动波浪、圆点图案、色块拼接
- 8 种配色方案：暖色调、冷色调、暗色调、典雅、鲜艳、黑白、自然、日落
- 实时预览，可自定义标题文字
- 支持保存到当前目录或指定子文件夹

### 2. 文章排版转换
- 将 Markdown 转换为适配微信公众号的 HTML（全内联样式）
- 4 款精心设计的主题：经典、优雅、现代、简约
- 自动将外部链接转为文末脚注（微信不支持外链跳转）
- 支持：标题、段落、粗体/斜体、代码块、引用、列表、表格、图片、分割线等
- 一键复制排版后的 HTML

### 3. 发布到公众号
- 自动上传文章中的本地图片到微信服务器
- 支持自动生成封面或选择本地图片作为封面
- 将文章发送到公众号草稿箱
- 支持设置标题、作者、摘要等元数据
- 从 Frontmatter 自动提取标题、作者、描述

## 安装

### 手动安装
1. 下载本项目中的 `main.js`、`manifest.json`、`styles.css` 文件
2. 在 Obsidian Vault 的 `.obsidian/plugins/` 目录下创建 `obsidian-mp-publisher` 文件夹
3. 将下载的文件放入该文件夹
4. 重启 Obsidian，在设置中启用插件

### 从源码构建
```bash
npm install
npm run build
```

## 使用

### 配置
在 Obsidian 设置 → 第三方插件 → MP Publisher 中配置：
- **AppID / AppSecret**：微信公众号的应用凭证（发布功能必需）
- **默认作者**：文章的默认作者名
- **默认主题**：Markdown 排版的默认样式
- **封面设置**：默认风格和配色

### 命令

| 命令 | 快捷方式 | 说明 |
|------|---------|------|
| `MP Publisher: 生成封面图` | — | 打开封面图生成器 |
| `MP Publisher: 预览文章排版` | — | 预览排版后的文章效果 |
| `MP Publisher: 复制排版后的 HTML` | — | 复制文章 HTML 到剪贴板 |
| `MP Publisher: 发布到公众号草稿箱` | — | 上传文章到微信公众号草稿箱 |

也可以通过左侧工具栏的上传图标快速进入发布流程。

### Frontmatter 支持

在文章开头使用 YAML frontmatter 设置元数据：

```yaml
---
title: 文章标题
author: 作者名
description: 文章摘要
---
```

如果未提供 frontmatter，插件会自动从文章内容中提取标题和摘要。

## 微信公众号配置

使用发布功能前，需要在微信公众号后台获取 AppID 和 AppSecret：

1. 登录[微信公众平台](https://mp.weixin.qq.com)
2. 进入 **设置与开发 → 基本配置**
3. 记录 AppID 和 AppSecret
4. 在 **设置与开发 → 基本配置 → IP白名单** 中添加你的 IP 地址
5. 在插件设置中填写 AppID 和 AppSecret

> **注意**：AppSecret 只在本地存储，请确保你的设备安全。插件仅将文章发送到草稿箱，不会直接发布。

## 文章主题预览

### 经典 (Default)
蓝色主色调，二级标题白字蓝底，引用块蓝色左边框。

### 优雅 (Elegant)
衬线字体，棕色系配色，首行缩进，适合文学类内容。

### 现代 (Modern)
渐变色标题，圆角卡片引用块，紫蓝渐变配色。

### 简约 (Simple)
极简设计，灰色系配色，最大化留白。
